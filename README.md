# Kiri Portfolio

A minimal, dark-themed personal portfolio. Static HTML and CSS -- no build step, no dependencies.

## Structure

```
index.html          Landing page
css/style.css       Styles
img/                Icons (GitHub, Roblox, Projects)
```

## Development

Open `index.html` in a browser. No server required.

## Deploy

Drop into any static host (Vercel, Netlify, GitHub Pages, etc).
